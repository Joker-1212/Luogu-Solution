## 题目描述

已知正整数$n$是两个不同的质数的乘积，试求出两者中较大的那个质数。

## 输入格式

一个正整数$n$。

## 输出格式

一个正整数$p$，即较大的那个质数。

## 样例 #1

### 样例输入 #1

```txt
21
```

### 样例输出 #1

```txt
7
```

## 提示

$n\le 2\times 10^9$

NOIP 2012 普及组 第一题
